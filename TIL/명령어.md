# get start

- 리액트 프로젝트를 만드는 방법은 여러가지가 있다.  
  |명령어|기능|
  |:--:|--|
  | npm init react-app . | 프로젝트 생성(마침표는 현재 디렉토리에서 프로젝트를 만들겠다는 뜻) |  
  | npm init react-app <폴더 이름> |---|  
  | npm run start | 프로젝트 실행. 단순히 프로젝트만 실행하는게 아니라 파일을 수정하면 그걸 인식해서  
  | --- | 바로 반영해주는 기능도 있다. -> '개발 모드를 실행했다' 라고도 한다. |  
  | control + C | 개발 모드 종료 |
